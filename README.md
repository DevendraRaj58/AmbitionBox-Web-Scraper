# AmbitionBox Web Scraper

## Project Overview

This project demonstrates web scraping using Python.

The scraper collects publicly available company information from AmbitionBox and stores it in a structured Pandas DataFrame.

## Data Collected

- Company Name
- Ratings
- Reviews
- Salary Information

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook

## Project Workflow

1. Send HTTP requests
2. Parse HTML pages
3. Extract company information
4. Store data in Pandas DataFrame
5. Export results to CSV
